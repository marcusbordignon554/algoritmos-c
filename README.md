# 💻 Algoritmos em C

FAETERJ Projeto Acadêmico

Repositório contendo exercícios e implementações desenvolvidas durante os estudos da linguagem C.

## 📖 Sobre

Este projeto reúne algoritmos desenvolvidos para prática de lógica de programação, estruturas de dados e resolução de problemas.

## 🚀 Conteúdos

- Estruturas condicionais
- Estruturas de repetição
- Vetores e matrizes
- Funções
- Ponteiros
- Manipulação de arquivos
- Estruturas de dados básicas

## 🛠 Tecnologias

- Linguagem C
- GCC
- Git/GitHub

## 🎯 Competências Desenvolvidas

- Lógica de programação
- Algoritmos
- Resolução de problemas
- Programação estruturada
